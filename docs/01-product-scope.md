# Product Scope

## MVP goal
Build an iOS-first FB2 reader that supports local import, offline reading, reading progress restore, word translation, paragraph translation, translated word history, and saved words collection.

## Included in MVP
- FB2 import from iOS Files
- Local file persistence in app sandbox
- FB2 metadata parsing
- Cover extraction when present
- No-cover fallback UI
- Library list
- Book details
- Reader with TOC and progress restore
- Word translation
- Paragraph translation
- Saved words tab
- Translated words tab
- Local translation cache

## Not included
- EPUB
- Conversion
- Bookmarks
- Learning or flashcards
- Search
- Cloud sync
- Auth
- Analytics
